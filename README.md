## <img src="https://img.shields.io/badge/-OVERVIEW-00BFFF?style=flat-square" height="25"> About This Tool

**TikTok Data Toolkit** is a powerful TikTok data scraper built with **Python** and **BeautifulSoup**. It fetches complete user information, extracts social links, downloads profile pictures, and supports bulk scraping — all with a hacker-style terminal interface optimized for Termux.
