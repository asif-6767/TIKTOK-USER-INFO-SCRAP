## <img src="https://img.shields.io/badge/-OVERVIEW-00BFFF?style=flat-square" height="25"> About TikTok Data Toolkit

**TikTok Data Toolkit** is a powerful, self-hosted TikTok data scraper built with **Python** and **BeautifulSoup**. It provides complete user information, social links extraction, profile picture download, and bulk scraping capabilities — all with a hacker-style terminal interface optimized for Termux.
